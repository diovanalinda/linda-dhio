SITEMAS OPERACIONAIS

1.1 Sistemas livres

Linux

<р> Linux é um sistema operacional de código aberto baseado no núcleo (kernel) Linux, criado por Linus Torvalds em 1991. É uma alternativa aos sistemas operacionais proprietários, como Windows e macOS. O Linux é conhecido por sua robustez, segurança e flexibilidade, o que o torna uma escolha popular para servidores, dispositivos móveis, sistemas embarcados e desktops. Linux, como sistema operacional, oferece uma base sólida e segura, enquanto as distribuições Linux permitem que usuários e empresas escolham versões que melhor atendam às suas necessidades específicas. A combinação do kernel Linux com uma variedade de softwares e ferramentas resulta em um ecossistema diversificado, adequado para uma ampla gama de aplicações.

Distribuições

<р> Uma distribuição Linux (ou distro) é uma versão do sistema operacional Linux que inclui o núcleo (kernel) do Linux, além de uma variedade de softwares e utilitários. Essas distribuições são criadas para atender a diferentes necessidades e objetivos, oferecendo pacotes que vão desde interfaces gráficas e ferramentas de desenvolvimento até aplicativos para servidores e dispositivos embarcados.

1.2 Sistemas proprietários

Apple

<р> Apple é conhecida por seus sistemas operacionais como macOS, iOS, watchOS e tvOS. Aqui estão algumas características dos sistemas operacionais da Apple: macOS: É o sistema operacional utilizado em computadores Mac. Conhecido por sua interface elegante e integração com outros dispositivos Apple, o macOS é amplamente utilizado por profissionais de design, edição de vídeo e programação. O macOS oferece uma experiência de usuário fluida e um ambiente seguro, com recursos como o Gatekeeper e o sandboxing para proteger o sistema de softwares maliciosos. iOS: O sistema operacional que roda em dispositivos móveis como iPhones e iPads. O iOS é famoso por sua segurança, desempenho e um ecossistema de aplicativos robusto disponível na App Store. O iOS é mais restritivo em comparação com o Android, limitando a personalização e a instalação de aplicativos de fontes externas. Segurança e Privacidade: A Apple enfatiza a segurança e a privacidade do usuário, implementando várias camadas de proteção em seus sistemas operacionais. Recursos como autenticação de dois fatores e criptografia de dados são comuns. Ambiente Fechado: Os sistemas da Apple são projetados para funcionar de forma otimizada com hardware Apple, criando um ecossistema fechado que garante uma experiência de usuário consistente, mas limita opções de personalização.

Microsoft

<р> Microsoft é a desenvolvedora do sistema operacional Windows, amplamente utilizado em desktops e laptops em todo o mundo. Algumas características dos sistemas operacionais da Microsoft incluem: Windows: A versão mais popular é o Windows 10, que se tornou um padrão para muitos ambientes de trabalho e pessoais. O Windows é conhecido por sua interface amigável, compatibilidade com uma ampla gama de hardware e suporte a uma vasta biblioteca de software. A Microsoft também oferece um modelo de atualização contínua, com novas funcionalidades e melhorias sendo lançadas regularmente. Integração com Outros Produtos: O Windows integra-se bem com outros serviços e produtos da Microsoft, como o Microsoft Office e o OneDrive, promovendo uma experiência coesa para usuários que utilizam múltiplos serviços. Segurança: Embora o Windows tenha historicamente enfrentado críticas por sua segurança, a Microsoft implementou várias medidas para proteger os usuários, como o Windows Defender, atualizações automáticas de segurança e ferramentas de controle de conta de usuário. Licenciamento e Custos: O Windows é um software pago, exigindo que os usuários adquiram uma licença para utilizá-lo. Isso pode ser uma desvantagem em comparação com sistemas operacionais de código aberto, como o Linux.

2 SOFTWARES

<р>O termo software refere-se ao conjunto de programas e instruções que dizem a um computador como realizar tarefas específicas. Em resumo, é o que permite que o hardware execute funções e ofereça utilidade ao usuário. Tipos de Software Os softwares podem ser classificados em várias categorias: Software de Sistema: Este tipo gerencia os recursos do hardware e fornece uma plataforma para outros softwares. Exemplos incluem sistemas operacionais como Windows, macOS e Linux. Software Aplicativo: Projetado para ajudar os usuários a realizar tarefas específicas, como processamento de texto (Microsoft Word), planilhas (Excel) e navegadores da web (Google Chrome). Software de Desenvolvimento: Ferramentas que ajudam os programadores a criar, testar e manter outros softwares, como ambientes de desenvolvimento integrado (IDEs) e compiladores. Software de Banco de Dados: Gerencia e organiza dados, permitindo que usuários e aplicativos acessem e manipulem informações. Exemplos incluem MySQL e Oracle. Software de Segurança: Protege o sistema contra vírus e outras ameaças. Exemplos incluem antivírus e firewalls. O software é fundamental para o funcionamento de computadores e dispositivos, permitindo que o hardware execute tarefas úteis e que os usuários interajam com a tecnologia de maneira eficaz.

2.1 Tipos de licenças

Licenciados

<р>Licenças de Software Licenciado (Proprietário) Definição: Este tipo de licença impõe restrições sobre como o software pode ser utilizado e distribuído. O código-fonte não está disponível para o público, e os usuários precisam adquirir uma licença para usar o software. Características: Uso Restrito: Os usuários geralmente não podem modificar, copiar ou redistribuir o software. Suporte e Atualizações: Muitas vezes, a licença inclui suporte técnico e atualizações periódicas. Exemplos: Windows, Microsoft Office, Adobe Photoshop. Vantagens: Suporte Técnico: Geralmente, as empresas oferecem suporte direto. Facilidade de Uso: Muitas vezes, essas aplicações têm interfaces amigáveis e são bem documentadas. Desvantagens: Custo: O software licenciado geralmente é pago. Falta de Flexibilidade: Limitações em personalizações e modificações.

Livres

<р> Licenças de Software Livre Definição: Softwares licenciados como "livres" permitem que os usuários executem, modifiquem e distribuam o software. A liberdade aqui se refere à liberdade de uso, e não necessariamente ao custo. Características: Código Aberto: O código-fonte é disponibilizado, permitindo modificações por qualquer pessoa. Distribuição Livre: Os usuários podem compartilhar o software com outras pessoas. Exemplos: Linux, LibreOffice, GIMP. Vantagens: Personalização: Os usuários podem adaptar o software para atender suas necessidades. Sem Custos: Muitas vezes, esses softwares são gratuitos. Comunidade: Geralmente, há uma comunidade ativa para suporte e desenvolvimento. Desvantagens: Suporte Limitado: O suporte pode depender da comunidade e não ter a mesma garantia de um software proprietário. Curva de Aprendizado: Pode ser mais desafiador para novos usuários, dependendo da distribuição ou aplicativo.

2.1.3 Propietários

<р>Software proprietário é aquele cujo código-fonte não é disponibilizado ao público. Isso significa que apenas a empresa ou o desenvolvedor que criou o software tem o direito de modificar, distribuir e vender o programa. Os usuários precisam adquirir uma licença para utilizar o software, e essa licença geralmente impõe restrições quanto ao uso e à redistribuição. O software proprietário desempenha um papel importante no mercado de tecnologia, oferecendo soluções robustas e suporte ao usuário. No entanto, a falta de acesso ao código-fonte e as restrições de licenciamento podem ser desvantagens para usuários que buscam maior liberdade e personalização em suas ferramentas.

2.2 Tipos de softwares de Sistemas

Aplicativos de Programação

<р> Aplicativos de programação são ferramentas que permitem aos desenvolvedores escrever, testar e depurar código. Eles incluem ambientes de desenvolvimento integrado (IDEs), editores de texto e ferramentas de versionamento. Essas aplicações facilitam a criação de software e a colaboração entre programadores. Exemplo: Visual Studio Code, um editor de código que suporta várias linguagens e possui extensões para funcionalidades adicionais.

Software para diagnóstico

<р> Refere-se a programas que analisam o desempenho de sistemas, identificando problemas e falhas. Eles ajudam a monitorar tanto hardware quanto software, oferecendo relatórios que ajudam na resolução de problemas. Exemplo: HWMonitor, que fornece informações sobre a temperatura e a voltagem de componentes de hardware.

Diagnóstico de software

<р>Este tipo de software é projetado para identificar e corrigir erros em aplicativos e sistemas operacionais. Ele pode incluir ferramentas que monitoram o desempenho e relatam falhas, ajudando desenvolvedores e usuários a resolver problemas específicos. Exemplo: Debugging tools, como o GDB, que permite depurar programas em várias linguagens de programação.

Diagnóstico de hardware

<р>Refere-se a ferramentas que avaliam o funcionamento de componentes físicos do computador, como processadores, memória e discos rígidos. O diagnóstico de hardware pode incluir testes de desempenho, identificação de falhas e verificações de temperatura. Exemplo: MemTest86, um programa que testa a memória RAM para detectar erros.

Ferramentas de Busca e Download

<р>Essas ferramentas ajudam os usuários a encontrar e baixar arquivos da internet. Elas podem incluir motores de busca e gerenciadores de download que facilitam a obtenção de conteúdo online. Exemplo: JDownloader, que permite gerenciar downloads de vários sites de compartilhamento.

Plugins

<р>Plugins são extensões que adicionam funcionalidades a um software existente. Eles permitem personalizar e expandir as capacidades do software sem precisar alterar seu código-fonte. Exemplo: Plugins para WordPress, que podem adicionar recursos como formulários de contato, SEO, entre outros.

Complementos do navegador

<р> Semelhantes aos plugins, os complementos do navegador são extensões que melhoram ou alteram as funcionalidades do navegador da web. Eles podem adicionar recursos como bloqueadores de anúncios ou ferramentas de produtividade. Exemplo: Grammarly, um complemento que ajuda a corrigir erros gramaticais e de estilo durante a escrita online.

Filtros de buscal

<р>Ferramentas que permitem aos usuários restringir ou refinar resultados de busca em motores de busca ou sistemas de gerenciamento de informações. Os filtros ajudam a localizar informações mais específicas de forma eficiente. Exemplo: Filtros em Google, que permitem refinar resultados por data, tipo de arquivo ou local.
